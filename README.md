# IID 激活服务 Cloudflare Worker
基于 Cloudflare Workers 构建的 IID 验证与请求日志服务，支持密码保护、分页查询、JSON 详情查看。

## 版本说明
works1.js 无日志
works2.js 带日志

## 带日志版本配置说明

### 1. 环境变量配置
在 Workers → 设置 → 变量 → 环境变量 中添加：

| 变量名 | 说明 | 默认值 | 是否必填 |
|--------|------|--------|----------|
| `LOG_PASSWORD` | `/logs` 日志页面登录密码 | — | ✅ 是 |
| `COOKIE_EXPIRE_HOURS` | 登录 Cookie 过期时间（小时） | 24 | ❌ 否 |
| `PAGE_SIZE` | 日志列表每页显示条数 | 20 | ❌ 否 |

### 2. KV 命名空间绑定
在 Workers → 设置 → KV 命名空间绑定 中添加：

| 变量名称（绑定名称） | 用途 | 是否必填 |
|----------------------|------|----------|
| `KV_LOGS` | 存储请求日志数据 | ✅ 是 |

> 注意：`KV_LOGS` 是**绑定名称**，不是环境变量，必须先创建 KV 再绑定。

## 访问地址
- 接口请求：`/`（POST）
- 日志后台：`/logs`（需密码登录）

## 功能特性
- ✅ DPoP 签名请求
- ✅ 请求日志持久化存储
- ✅ 密码保护 + Cookie 自动过期
- ✅ IID 关键字搜索
- ✅ 分页展示
- ✅ 单条日志 JSON 详情查看
- ✅ 跨域支持
